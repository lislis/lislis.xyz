Sound memory exists because the world of MIDI controllers offers wonderfully creative interfaces. The Launchpad Mini used in this work mainly consists of a simple grid layout, which immediately reminded me of game "Memory" that I played as a child. I loved laying out the cards in a strict grid (Not just in a pile! Savages!) It made it easier to remember which cards had already been turned over and moved, they would differ from my strict grid and give me an advantage.

So I remade the Launchpad into a game of memory. Instead of using the controller as just an input for audio software, it is now the playgroud itself. The programming of lights allows for two players taking turns finding matching pairs of animal sounds.

The Launchpad's buttons are very haptic, the digital display used to control the game logic very minimal. This could be used as an opportunity to play the game together with visually impaired people or children who are not allowed a lot of screen time yet.

I'm also working on a setup that can run from a smartphone.
